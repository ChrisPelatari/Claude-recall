Privacy Policy

Last updated: 2026-05-20

Short version: Claude Recall runs entirely on your device. It does not collect, transmit, or share any personal data, file content, or usage telemetry. There is no server, no analytics, no tracking.

1. Scope

This policy covers Claude Recall — the native macOS & iOS app distributed via GitHub Releases.

Claude Recall is a fork of nvwalj/ai-memory-reader. Upstream ships a companion /aimr Claude Code plugin at nvwalj/aimr-claude-plugin; that plugin has its own privacy policy in the upstream project and is not covered here.

2. What data the app handles

Claude Recall is a read/edit tool for markdown files that already live on your local device or filesystem (such as CLAUDE.md, MEMORY.md, or files in ~/.claude/, ~/.openclaw/, etc.). File content is read directly from disk and rendered locally.

3. What data the app sends over the network

Effectively none.

  • The app has no analytics, no crash reporting service, no telemetry endpoints, and no remote configuration.
  • The only outbound request the app may make on its own is a once-per-24-hour poll of api.github.com to check whether a newer release is available. This is skippable from the Help menu and is disabled on Mac App Store builds (where Apple handles updates).
  • The app may also fetch images explicitly embedded in markdown you open (e.g. ![](https://example.com/foo.png)). This is the same behavior as any markdown renderer and is initiated by content you choose to view.

4. What data we collect about you

None. The project does not run a backend service. There are no user accounts. We do not see, store, or process any information about your device, files, prompts, AI agent memory, or usage patterns.

5. Third-party services

  • GitHub hosts the source code, releases, and this website. GitHub's own privacy policy applies when you download a release or view this page: GitHub privacy statement.
  • Ko-fi — the upstream author accepts optional donations at ko-fi.com/nvwalj. Donations are entirely optional and handled by Ko-fi and its payment processors. The app itself does not contact Ko-fi. See Ko-fi's privacy policy.

6. Permissions the app requests

  • File access — required to open and render the markdown files you choose. Files are read on demand; nothing is uploaded.
  • URL scheme handler registration — so that clauderecall:// links (used by the crec CLI and other agents) can open the app. Handling a URL scheme does not transmit data anywhere.

7. Children

This product is a general-purpose developer utility and is not directed to children under 13. We do not knowingly collect data from anyone.

8. Changes

If this policy ever changes, the updated text will appear at this URL and the "Last updated" date above will change. Material changes will also be noted in the GitHub release notes.

9. Contact

Questions or concerns: open an issue at github.com/ChrisPelatari/Claude-recall/issues.
